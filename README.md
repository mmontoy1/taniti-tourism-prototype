## Project Overview

This project is a redesigned tourism website for the island of Taniti.  
The goal of this project is to improve user experience, navigation clarity, accessibility, and mobile responsiveness while increasing tourism engagement.

The website was developed as part of a UX design process including:

- Timeline planning
- Persona development
- Wireframe creation
- Guerrilla usability testing
- Prototype refinement

---

## Features

- Responsive design
- Clear navigation structure
- Attractions listing with booking options
- Lodging and dining information
- Transportation details
- Travel information page
- Functional booking form with confirmation message
